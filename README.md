# ping_pong
Game Ping Pong Menggunakan Pygame

Ini adalah game multiplayer yang bisa dimainkan 2 orang.

Untuk pemain kiri (pertama) digerakkan menggunakan tombol W dan tombol S
Untuk pemain kanan (kedua) digerakkan menggunakan tombol panah atas dan tombol panah bawah

Untuk memenangkan game bola harus menyentuh garis lawan.
